# Machine Learning Algorithms From Scratch

This repository contains implementations of classic Machine Learning algorithms built from scratch using only NumPy (no high-level ML libraries such as scikit-learn).

The goal of this project is to deeply understand the mathematical foundations and internal mechanics of each algorithm.
